### 教程特点

- **七科全覆盖**：数据结构（7 章）、计算机组成原理（7 章）、操作系统（5 章）、计算机网络（7 章）、高等数学（8 章 ✅ 部分完成）、线性代数（1 章）、概率统计（1 章），覆盖 408 + 数一全部知识点
- **考研导向**：每章标注考情提示（分值/题型）、技术生命周期标记（🔥 广泛应用 / ⚠️ 逐渐淘汰 / 💀 过时）、常见题型汇总
- **纯前端静态文件**：基于 VitePress 构建，无需任何后端服务，可部署到 GitHub Pages、nginx 或任意静态托管
- **轻量编辑**：所有内容按章节拆分为独立 Markdown 文件，每个文件聚焦一个专题，编辑轻量、构建飞快
