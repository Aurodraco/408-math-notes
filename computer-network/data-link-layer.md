# 数据链路层

## 数据链路层的功能

- 为网络层提供服务（无确认无连接 / 有确认无连接 / 有确认面向连接）
- 链路管理
- 帧定界、帧同步、透明传输
- 流量控制
- 差错控制

## 组帧

- 字符计数法
- 字符填充的首尾定界法
- 比特填充的首尾定界法（零比特填充）
- 违规编码法

## 差错控制

### 检错编码

- 奇偶校验码
- 循环冗余校验（CRC）

### 纠错编码

- 海明码（发现双比特错，纠正单比特错）

## 流量控制与可靠传输

### 停止-等待协议

- 基本原理
- 信道利用率低

### 后退 N 帧协议（GBN）

- 累计确认
- 发送窗口大小：1 ≤ W_T ≤ 2^n - 1

### 选择重传协议（SR）

- 逐帧确认
- 收发窗口大小：W_T = W_R ≤ 2^(n-1)

## 介质访问控制（MAC）

### 信道划分

- **频分多路复用（FDM）**
- **时分多路复用（TDM）**
- **波分多路复用（WDM）**
- **码分多路复用（CDM / CDMA）**

### 随机访问

- **ALOHA 协议**（纯 ALOHA / 时隙 ALOHA）
- **CSMA**：1-坚持、非坚持、p-坚持
- **CSMA/CD**（以太网）
  - 争用期（2τ）
  - 最短帧长
  - 二进制指数退避算法
- **CSMA/CA**（无线局域网）
  - RTS/CTS 握手机制

### 轮询访问

- 令牌传递协议

## 局域网

### 以太网

- MAC 帧格式（目的地址、源地址、类型、数据、FCS）
- MAC 地址（6 字节）
- 高速以太网（100M、1G、10G）

### 无线局域网

- IEEE 802.11
- 帧格式（四个地址字段）

### VLAN

- 将物理 LAN 划分为多个逻辑 LAN

## 广域网协议

### PPP 协议

- 帧格式
- LCP + NCP
- 透明传输（字节填充 / 比特填充）

## 数据链路层设备

### 网桥

- 透明网桥（自学习）
- 生成树算法（STP）

### 交换机

- 多端口网桥
- 交换方式（直通式 / 存储转发式）
- 自学习算法（转发表）

## 常见题型

- CSMA/CD 最短帧长计算
- GBN / SR 协议窗口大小的计算
- 交换机转发表的更新过程
- CRC 校验码的计算
