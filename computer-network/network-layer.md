## IPv4

### IPv4 地址

- 分类 IP 地址（A、B、C、D、E 类）
- 特殊 IP 地址（网络号/主机号全0、全1、环回地址）
- 私有 IP 地址范围

### 子网划分

- 子网掩码
- 子网地址计算
- 子网中可用的主机数

### CIDR（无分类编址）

- 斜线记法（a.b.c.d / n）
- 地址块、超网
- 最长前缀匹配

### ARP 协议（地址解析协议）

- IP 地址 → MAC 地址
- ARP 请求/应答
- ARP 缓存

### DHCP 协议（动态主机配置）

- IP 地址自动分配
- 租用期

### ICMP 协议（网际控制报文）

- ICMP 差错报告报文（终点不可达、时间超过、参数问题、改变路由）
- ICMP 询问报文（回送请求/应答、时间戳请求/应答）
- PING 和 Traceroute 的原理

## 路由协议

### 自治系统（AS）

### 内部网关协议（IGP）

#### RIP（距离向量路由）

- 基于距离向量
- 跳数为度量
- 慢收敛问题（好消息传得快，坏消息传得慢）
- 最大跳数 15

#### OSPF（链路状态路由）

- 基于链路状态
- 使用 Dijkstra 最短路径算法
- 多层次（主干区域 + 其他区域）
- Hello 协议维持邻接关系

### 外部网关协议（EGP）

#### BGP（边界网关协议）

- 路径向量协议
- 基于策略，而非距离

### 三种路由协议的对比

| 协议 | 类型 | 算法 | 封装协议 |
|------|------|------|----------|
| RIP | 内部 | 距离-向量 | UDP |
| OSPF | 内部 | 链路状态 | IP |
| BGP | 外部 | 路径-向量 | TCP |
