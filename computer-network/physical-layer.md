# 物理层

> **考情提示**：物理层在 408 中一般 1 道选择题（2 分）。重点：奈奎斯特/香农公式的计算、曼彻斯特编码波形、PCM 采样定理。公式套算纯送分——知道如何代入即可。

## 通信基础

### 基本概念

- **数据**：运送消息的实体。**信号**：数据的电气或电磁表现。
- **模拟信号 / 数字信号**：模拟信号连续、数字信号离散（0‒1跳变）。电话线传模拟信息，计算机产生数字信号。
- **码元**：一个固定时长的信号波形——表示一个 k 位二进制数。两个码元表示 1 位值（k=1 时为普通的双态）。
- **通信方式**：
  - **单工**：数据只能沿一个方向传送（如无线电广播）。
  - **半双工** 🔥：可双向通信但在一个时刻只能一个方向传送（如对讲机）。
  - **全双工** 🔥：同时双向通信（如电话、网络通信）。

### 信道的极限容量

信道的极限容量由**奈奎斯特**和**香农**两个定理从不同角度描述 🔥：

**奈奎斯特定理（无噪声信道）** 💀：
> 最大码元传输速率 = 2W （波特/baud），其中 W 是信道带宽（Hz）。
>
> 最大数据传输率 = 2W × log₂V （bps），其中 V 是码元携带的离散电平数。

此定理告诉我们在无噪声理想信道中，码元速率被带宽限制，但通过增加每码元可表示的比特数（多电平编码）可以不断提高数据率。

**香农定理（有噪声信道）** 🔥 🔥：
> 最大数据传输率 = W × log₂(1 + S/N) （bps）

其中 S/N 是信号与噪声的功率比（信噪比）。在现实的所有信道，噪声决定了信息传输的上限——**香农定理是绝对上限，没有技术能超越它**。

**信噪比常用分贝（dB）表示 🔥**：`dB = 10 × log₁₀(S/N)`。例如 30dB 表示 S/N = 1000（10^(30/10)）。

> 408 经典考法：给带宽 W 和信噪比 dB 值，先转换 dB→S/N 比值，代入香农公式得最大数据率；同时判断奈奎斯特速率可能大于香农公式但**真正的上线是香农值**（因为噪声存在）。这道题每年反复出现。

### 编码与调制

**数字数据 → 数字信号（基带调制）** 🔥：

| 编码方式 | 原理 | 特点 |
|----------|------|------|
| **NRZ** | 不归零制——低电平 0 高电平 1 | 简单但无时钟同步能力 |
| **NRZI** | 不归零倒转——遇 1 电平翻转、遇 0 不变 | USB 使用。同步稍好，但长串 0 也会失去同步 |
| **曼彻斯特编码** 🔥 | 每个码元中间发生跳变——**上升沿为 1 下降沿为 0**（一种标准） | 自带时钟信号——每比特中间一定有跳变——以太网使用 🔥🔥 |
| **差分曼彻斯特编码** | 每码元起始处：遇 0 翻转、遇 1 保持——码元中间仍然跳变 | 更强的抗干扰时钟同步能力 |

> 408 必考题：给你一个比特串，要求画出曼彻斯特编码或差分曼彻斯特的波形——核心是掌握「中间一定要有跳变」的原则。

**数字数据 → 模拟信号（调制）** 🔥：
- **ASK（幅移键控）**：用不同幅度表示 0 和 1——易受噪声干扰。
- **FSK（频移键控）**：用不同频率表示 0 和 1——抗干扰性较好。
- **PSK（相移键控）** 🔥：用相位的变化表示数据（如 0° 为 0、180° 为 1）。Wi-Fi 广泛使用 PSK 变种。
- **QAM（正交幅度调制）** 🔥 🔥：同时改变幅度和载波的相位——每个码元表示多个比特——现代高速网络的标准。

**模拟数据 → 数字信号（PCM 脉码调制）** 🔥：分三步：① **采样**：按 ≥2 倍最高信号频率的速率获取信号电压值——这是**奈奎斯特采样定理**（考研常考采样速率的计算）。② **量化**：将采样值从连续范围映射为有限离散整数。③ **编码**：把量化的整数替换为二进制编码。

## 传输介质

### 导向性传输介质

- **双绞线** 🔥：两根互相绕拧的铜线降低电磁干扰和串扰。分 UTP（无屏蔽）和 STP（有屏蔽）。是以太网标准 100BASE-T、1000BASE-T 的标配介质。3/5/5e/6 类线的差别在于缠绕精度和最高支持频率。
- **同轴电缆** 💀：外有铜网屏蔽内部信号导体——比双绞线抗干扰强、传输距离长。曾广泛用于有线电视和早期以太网——现在被双绞线和光纤替代。
- **光纤** 🔥 🔥：利用全反射原理在极细光纤维中传送光信号。分**多模光纤**（芯粗—多条光不同角度反射——短距经济）和**单模光纤**（芯极细仅一条路径传输—长距跨都市和海底电缆——高带宽）。光纤不受电磁干扰、损耗极低、带宽超高频——是现代通信网络的主串线路。

### 非导向性传输介质

- **无线电波** 🔥：穿透能力强、传播远——Wi-Fi（2.4/5GHz）和蜂窝通信依赖基地。
- **微波** 🔥：地面中继接力传播——短波径直线传播需中继塔，受雨雪影响（会议卫星通信的变体是长距微波落地）。🔥 移动 5G 网还使用微波频段。
- **红外线 / 激光** ⚠️：短距离直线传输——可进行点对点的屋顶建筑间链路但不普遍。

## 物理层设备

### 中继器

中继器 💀 工作在物理层——将衰弱变形的输入信号重生为原始强度与形状再传出。号数增加网络段的最长距离但不能延伸任何网络逻辑。两段用中继器连接的以太网形成同一个冲突域——过多中继累积迟延触错以太网规范限制。

### 集线器（Hub）

集线器（Hub）💀 就像一个多端口的中继器——数据从任何一个端口进入均被广播到其他所有端口。所有端口共享带宽，所有接入设备在一个冲突域内。半双工方式。集线器在现今网络中几乎完全被**交换机**取代 🔥（交换机的全双工无碰撞工作方式见数据链路层）。

> 考研区分：「集线器属于【物理层/数据链路层】？」——物理层；「交换机属于？」——数据链路层。

## 常见题型

- **奈奎斯特/香农公式的计算**：给 dB 噪比→先换算比值→代入香农公式得出最大速率。注意两公式同时存在——香农值才是实际上下限。
- **曼彻斯特/差分曼彻斯特编码的波形绘制**：给比特串，画出每个脉冲周期的跳变方向（先高后低 = 0 / 先低后高 = 1 标准一版本）。注意差分——先前的高低决定本次的起始电平。
- **PCM 采样频率计算**：给定模拟信号的最高频率 fmax→采样速率至少 2×fmax 才可避免信号混淆（Nyquist 采样定理）。
