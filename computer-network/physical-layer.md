# 物理层

## 通信基础

### 基本概念

- 数据、信号、码元
- 模拟信号 / 数字信号
- 单工、半双工、全双工通信

### 信道的极限容量

- 奈奎斯特定理（无噪声信道）：最大码元传输速率 = 2W（baud）
- 香农定理（有噪声信道）：最大速率 = W × log₂(1 + S/N)
- 信噪比（dB）：10 × log₁₀(S/N)

### 编码与调制

- 数字数据 → 数字信号：NRZ、NRZI、曼彻斯特、差分曼彻斯特
- 数字数据 → 模拟信号：ASK、FSK、PSK、QAM
- 模拟数据 → 数字信号：PCM（采样 → 量化 → 编码）

## 传输介质

### 导向性传输介质

- 双绞线
- 同轴电缆
- 光纤（多模 / 单模）

### 非导向性传输介质

- 无线电波
- 微波
- 红外线、激光

## 物理层设备

### 中继器

- 放大信号、扩大网络范围
- 不能连接不同协议的网络

### 集线器（Hub）

- 多端口中继器
- 共享带宽，半双工

## 常见题型

- 奈奎斯特/香农公式的计算
- 曼彻斯特编码的波形绘制
- PCM 采样频率计算（≥ 2倍最高频率）
