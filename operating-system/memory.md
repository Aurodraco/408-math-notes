### 页面置换算法

| 算法 | 要点 |
|------|------|
| 最佳置换（OPT）| 无法实现，用于衡量其他算法 |
| 先进先出（FIFO）| Belady 异常 |
| 最近最久未使用（LRU）| 堆栈类算法，无 Belady 异常 |
| 时钟置换（CLOCK/NRU）| 访问位 + 修改位 |
| 改进型 CLOCK | 四类页面（0,0)(0,1)(1,0)(1,1) |
