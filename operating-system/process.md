### 调度算法

| 算法 | 类型 | 抢占？ | 特点 |
|------|------|--------|------|
| 先来先服务（FCFS） | 作业/进程 | 否 | 有利于长作业 |
| 短作业/短进程优先（SJF/SPF） | 作业/进程 | 均可 | 平均等待时间最短 |
| 高响应比优先（HRRN） | 作业 | 否 | 响应比 = (等待+服务)/服务 |
| 时间片轮转（RR） | 进程 | 是 | 时间片大小是关键 |
| 优先级调度 | 进程 | 均可 | 可能导致饥饿 |
| 多级队列 | 进程 | 是 | 不同队列不同策略 |
| 多级反馈队列 | 进程 | 是 | 综合最优 |
