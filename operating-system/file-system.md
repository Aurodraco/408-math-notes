## 文件系统的实现

### 文件的物理结构（文件的分配方式）

- **连续分配**：顺序存取快，增删困难，产生外碎片
- **链接分配**：
  - 隐式链接（FAT 要遍历）
  - 显式链接（FAT 表，支持随机存取）
- **索引分配**：
  - 单级索引
  - 多级索引
  - 混合索引（直接 + 间接块，Unix/Linux）

### 文件存储空间管理

- 空闲表法
- 空闲链表法（空闲盘块链 / 空闲盘区链）
- 位示图法（重点）
- 成组链接法（Unix/Linux）

## 磁盘组织与管理

### 磁盘的结构

- 盘面、磁道、扇区、柱面
- 磁盘的物理地址 CHS

### 磁盘调度算法

| 算法 | 要点 |
|------|------|
| 先来先服务（FCFS）| 公平，效率低 |
| 最短寻找时间优先（SSTF）| 可能饥饿 |
| 扫描算法（SCAN / 电梯） | 双向扫描 |
| 循环扫描（C-SCAN） | 单向扫描 |
| LOOK 和 C-LOOK | SCAN 的改进版 |

### 磁盘读写时间

- 寻道时间（大头）
- 旋转延迟时间
- 传输时间

### 磁盘管理

- 磁盘格式化
