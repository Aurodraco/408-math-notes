# 操作系统概述

## 操作系统的基本概念

### 操作系统的定义

- 控制和管理整个计算机系统的硬件和软件资源
- 合理地组织、调度计算机的工作与资源的分配
- 为用户和其他软件提供方便的接口和环境

### 操作系统的特征

- 并发
- 共享（互斥共享、同时共享）
- 虚拟
- 异步

### 操作系统的目标和功能

- 资源管理（处理机、存储器、设备、文件）
- 作为用户与计算机硬件之间的接口
- 实现了对计算机资源的扩充
