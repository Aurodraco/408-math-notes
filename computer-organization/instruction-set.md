# 指令系统

## 指令格式

### 指令的基本格式

- 操作码字段 + 地址码字段
- 定长操作码 vs 扩展操作码

### 地址码的个数

- 零地址指令
- 一地址指令
- 二地址指令
- 三地址指令

### 扩展操作码设计

- 操作码位数随地址数减少而增长
- 各类型指令条数的计算

## 指令的寻址方式

### 指令寻址

- 顺序寻址（PC + 1）
- 跳跃寻址（JMP / CALL / RET）

### 数据寻址

- **隐含寻址**：操作数在累加器中
- **立即寻址**：指令直接给出操作数
- **直接寻址**：指令给出的地址即操作数有效地址
- **间接寻址**：一次 / 多次间接
- **寄存器寻址**：操作数在寄存器中
- **寄存器间接寻址**：寄存器中存操作数地址
- **相对寻址**：EA = PC + A（用于转移指令）
- **基址寻址**：EA = 基址寄存器 + A（用于程序重定位）
- **变址寻址**：EA = 变址寄存器 + A（用于数组访问）
- **堆栈寻址**：操作数在栈顶

### 基址寻址与变址寻址的对比

| 对比项 | 基址寻址 | 变址寻址 |
|--------|----------|----------|
| 目的 | 程序重定位 | 数组/循环访问 |
| 谁变谁不变 | 基址寄存器不变，A 变 | 变址寄存器变，A 不变 |

## CISC 和 RISC

### CISC（复杂指令集计算机）

- 指令数目多、长度不等、寻址方式多样
- 微程序控制
- 典型：x86

### RISC（精简指令集计算机）

- 指令数目少、长度定长、寻址方式简单
- 硬布线控制（组合逻辑）
- 流水线效率高
- 典型：ARM、MIPS、RISC-V
