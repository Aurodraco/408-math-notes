# 中央处理器

## CPU 的功能和基本结构

### CPU 的功能

- 指令控制、操作控制、时间控制、数据加工、中断处理

### CPU 的基本结构

- 运算器和控制器
- 通用寄存器组、专用寄存器（PC、IR、MAR、MDR、PSW）

### 指令周期

- 取指周期 → 间址周期 → 执行周期 → 中断周期

### 数据通路

- 单总线结构、多总线结构、专用数据通路
- 数据通路的时序控制

### 指令执行的数据通路分析

- 取指令
- 运算类指令（ADD、SUB 等）
- 访存类指令（LOAD、STORE）
- 转移类指令（JMP）

### 硬布线控制器（组合逻辑）

## 指令流水线

### 指令流水线的基本概念

### 流水线的性能指标

- 吞吐率
- 加速比
- 效率

### 影响流水线的因素

- **结构相关**：资源冲突 → 增加硬件资源 / 暂停
- **数据相关**：数据冲突（RAW、WAR、WAW）→ 旁路技术 / 暂停
- **控制相关**：转移指令 → 分支预测 / 延迟槽

### 流水线的分类

### 超标量流水线

- 多发射、动态调度

## 异常和中断

### 内部异常和外部中断

- 内中断（异常）：故障、自陷、终止
- 外中断：可屏蔽中断、不可屏蔽中断

### 中断处理过程

- 中断请求 → 中断判优 → 中断响应 → 中断服务 → 中断返回
- 中断向量表
- 保存和恢复现场
