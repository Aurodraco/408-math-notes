## 高速缓冲存储器（Cache）

### Cache 的基本工作原理

- 地址映射
- 写策略

### Cache 与主存的映射方式

- **直接映射**：Cache 行号 = 主存块号 mod Cache 行数
- **全相联映射**：主存块可放入 Cache 任意行
- **组相联映射**：组间直接映射，组内全相联映射

### 替换算法

- 随机（RAND）
- 先进先出（FIFO）
- 近期最少使用（LRU）
- 最不经常使用（LFU）

### 写策略

- 写命中：全写法（Write Through）、写回法（Write Back）
- 写不命中：写分配法、非写分配法

- 命中率、平均访问时间
- Cache 容量的影响
