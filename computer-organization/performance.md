# 计算机的性能指标与评测

## 综合性能指标

### CPU 性能公式

- CPU 时间 = 指令数 × CPI × 时钟周期
- 各因素的含义及优化方向

### Amdahl 定律

- 系统加速比公式：S = 1 / ((1 - f) + f / n)
- 改进部分 f 对整体性能的影响
- 互斥改进与叠加改进
