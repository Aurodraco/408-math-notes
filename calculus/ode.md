# 常微分方程

> **考情提示**：常微分方程在数一中通常 1 道选择题或填空题 + 1 道解答题（共约 8–14 分）。一阶方程和最核心的二阶常系数线性方程是考频最高的两类。方程类型的识别和套用对应解法模板是这章的决胜关键。

## 微分方程的基本概念

**微分方程**：含自变量、未知函数及其导数的方程。

**阶数**：方程中出现的最高阶导数的阶数。

**通解** 🔥：解中含与阶数同个数的**独立任意常数**。如二阶方程通解形式为 $y = \varphi(x, C_1, C_2)$。**特解**是通过附加初始条件确定了各常数后的具体解。

## 一阶微分方程

### 可分离变量方程

能写成 $\frac{dy}{dx} = f(x) g(y)$ 的形式 🔥。解法：分离积两边：

$$
\int \frac{dy}{g(y)} = \int f(x) \, dx + C
$$

> 考卷上最简单的 DE 题型——辨认出形式后纯粹是积分技术的测试。

### 齐次方程

形如 $\frac{dy}{dx} = \varphi\left(\frac{y}{x}\right)$ 🔥。解法：令 $u = \frac{y}{x}$（即 $y = ux$），则 $\frac{dy}{dx} = u + x\frac{du}{dx}$。代入原方程可得关于 $u$ 的可分离变量方程。

> 常见的辨认标志：分子分母的各项次数相同，或括号中全是 $\frac{y}{x}$ 组合。

### 一阶线性微分方程

标准形式 🔥：$\frac{dy}{dx} + P(x) y = Q(x)$

**通解公式**（必背）：

$$
y = e^{-\int P \, dx} \left[ \int Q \, e^{\int P \, dx} \, dx + C \right]
$$

**步骤口诀**：先求积分因子 $\mu = e^{\int P \, dx}$，再两边同乘 $\mu$，左边变为 $(y\mu)'$，积分移项得解。

> 这是一阶方程中考频最高的类型——识出 $P(x)$ 和 $Q(x)$ 后套公式即可。

### 伯努利方程

形如 $\frac{dy}{dx} + P(x) y = Q(x) y^n$（$n \neq 0, 1$）⚠️。解法：整理为 $y^{-n} \frac{dy}{dx} + P(x) y^{1-n} = Q(x)$，令 $z = y^{1-n}$，则化为关于 $z$ 的一阶线性方程。

## 高阶线性微分方程

### 线性微分方程解的结构

$n$ 阶线性齐次方程：$y^{(n)} + a_1 y^{(n-1)} + \cdots + a_n y = 0$

- 若 $y_1, y_2, \dots, y_n$ 是该方程的 $n$ 个**线性无关**解，则通解为 $y = C_1 y_1 + C_2 y_2 + \cdots + C_n y_n$

$n$ 阶线性非齐次方程：$y^{(n)} + a_1 y^{(n-1)} + \cdots + a_n y = f(x)$

- **通解 = 对应齐次方程的通解 + 非齐次方程的一个特解** 🔥

### 二阶常系数线性齐次方程

> 这是考研 ODE 考察频率最高的具体类型

标准形式：$y'' + py' + qy = 0$（$p, q$ 为常数）

**特征方程**：$\lambda^2 + p\lambda + q = 0$

判别式 $\Delta = p^2 - 4q$：

| 判别式 | 特征根 | 通解形式 🔥 |
|:--:|------|------|
| $\Delta > 0$ | 两个不等实根 $\lambda_1 \neq \lambda_2$ | $y = C_1 e^{\lambda_1 x} + C_2 e^{\lambda_2 x}$ |
| $\Delta = 0$ | 一个重根 $\lambda$ | $y = C_1 e^{\lambda x} + C_2 x e^{\lambda x}$ |
| $\Delta < 0$ | 共轭复根 $\alpha \pm i\beta$ | $y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$ |

### 二阶常系数线性非齐次方程

标准形式：$y'' + py' + qy = f(x)$

**设特解方法** 🔥：根据右端非齐次项 $f(x)$ 的形式直接尝试：

**类型一**：$f(x) = P_m(x) e^{\lambda x}$（多项式乘指数）

特解形式：$y^* = x^k Q_m(x) e^{\lambda x}$，其中 $k$ 为特征根 $\lambda$ 的重复次数（$k=0$ 若非根、$k=1$ 若单根、$k=2$ 若重根）

**类型二** 🔥：$f(x) = e^{\alpha x}[P_m(x) \cos\beta x + Q_n(x) \sin\beta x]$

特解形式：$y^* = x^k e^{\alpha x}[R_l(x) \cos\beta x + S_l(x) \sin\beta x]$，其中 $l = \max(m,n)$，$k$ 取决于 $\alpha \pm i\beta$ 是否为特征根

> 特解设好后代入方程，通过比较同类项系数确定待定系数——这是考研解答题计算量最大的步骤。

### 欧拉方程

形如 $x^2 y'' + a x y' + b y = f(x)$ 的方程 🔥（变量系数但有其专门解法）。

解法：令 $x = e^t$ → 将方程转化为常系数线性方程：

$$
x y' = \frac{dy}{dt}, \quad x^2 y'' = \frac{d^2 y}{dt^2} - \frac{dy}{dt}
$$

代入原方程标准化成常系数后再按上述方法求解。

## 微分方程的应用

考研应用题主要有两类：

1. **几何应用** 🔥：给曲线某点的切线/法线斜率 $\to$ 列出 $y' = f(x,y)$ 的方程求解
2. **物理应用**：牛顿冷却定律、放射性衰减、弹簧振动等——给出瞬时变化率与量的关系列出微分方程。常考一阶线性方程和二端振动二阶齐次方程

> 解答应用题的通用步骤：① 根据题意列出导数和量的关系→② 识别方程类型选用解法→③ 甩入初始条件定常数 $C$

## 常见题型

- **一阶方程类型识别**（选择/填空高频）：给一个方程，判断它是可分离/齐次/一阶线性的哪一类，选用对应方法
- **非齐次方程设特解形式**：给非齐次项和齐次部分特征根，直接写出特解形式不用解出系数（填空题反复出）
- **二阶常系数齐次通解**：特征根三种情况 → 代入对应通解结构（$\Delta>0/\Delta=0/\Delta<0$）
- **二阶常系数非齐次的完整求解**：齐次通解 + 特解叠加 $\to$ 初值代入求常数
- **欧拉方程**：代换 $x=e^t$ 将其变标准常系数方程再求解
