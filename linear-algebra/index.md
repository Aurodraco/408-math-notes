# 线性代数（上）— 行列式与矩阵

> **考情提示**：线性代数占数一总分约 22%（≈33 分），通常 2–3 道选择/填空 + 1 道解答题。行列式与矩阵是全部后续章节的工具基础——秩、逆矩阵、初等变换贯穿始终。本章偏重计算，熟练掌握矩阵运算公式即可拿稳基础分。

## 行列式

### $n$ 阶行列式的定义

$n$ 阶行列式是 $n!$ 个项的代数和，每项由不同行不同列的元素乘积配上排列逆序数的正负号组成：

$$
|A| = \det(A) = \sum (-1)^{\tau(p_1 p_2 \cdots p_n)} a_{1p_1} a_{2p_2} \cdots a_{np_n}
$$

> 💀 考研完全不要求用定义硬算 $n$ 阶行列式——所有计算通过性质和初等变换完成。

### 行列式的基本性质（必背）

| 性质 | 说明 |
|------|------|
| 转置不变 | $|A^T| = |A|$ |
| 互换两行（列） | 行列式变号 |
| 两行（列）相等 | 行列式 $= 0$ |
| 某行（列）全零 | 行列式 $= 0$ |
| 某行（列）乘以 $k$ | 行列式变为原来的 $k$ 倍 |
| 某行（列）的 $k$ 倍加到另一行 | 行列式值不变 🔥 |
| 若两行成比例 | 行列式 $= 0$ |
| 上/下三角行列式 | $=$ 主对角线元素乘积 |

### 行列式的计算

**展开定理** 🔥：按第 $i$ 行展开

$$
|A| = \sum_{j=1}^{n} a_{ij} A_{ij}, \quad A_{ij} = (-1)^{i+j} M_{ij}
$$

其中 $A_{ij}$ 为代数余子式，$M_{ij}$ 为余子式。

**初等变换法** 🔥：通过第三种初等行运算（某行的 $k$ 倍加到另一行）将行列式化为**上三角**，值 $=$ 主对角线乘积。

> 考试技巧：首选初等变换化三角行列式——步骤少、计算快、不易出错。只有在变换出现分数极复杂时再用展开法。

**范德蒙德行列式**：

$$
\begin{vmatrix}
1 & 1 & \cdots & 1 \\
x_1 & x_2 & \cdots & x_n \\
x_1^2 & x_2^2 & \cdots & x_n^2 \\
\vdots & \vdots & \ddots & \vdots \\
x_1^{n-1} & x_2^{n-1} & \cdots & x_n^{n-1}
\end{vmatrix}
= \prod_{1 \leq i < j \leq n} (x_j - x_i)
$$

> 只需识别出范德蒙行列式形式后直接背结论代入值——考察频率高但都不过是套公式。

## 矩阵

### 矩阵的基本运算

**加减法**：同型矩阵对应元素相加减。

**数乘**：$kA = (k \cdot a_{ij})$。

**乘法** 🔥：$(AB)_{ij} = \sum_{k=1}^{s} a_{ik} b_{kj}$——$A$ 的列数必须等于 $B$ 的行数，一般 $AB \neq BA$。

**转置**：$(A^T)_{ij} = A_{ji}$。

运算规律（可背）：

| 运算法则 | 是否成立 |
|------|:--:|
| $AB = BA$ | ❌ 一般不成立 |
| $(AB)C = A(BC)$ | ✅ 结合律 |
| $(AB)^T = B^T A^T$ | ✅ 逆转序 |
| $(AB)^{-1} = B^{-1} A^{-1}$ | ✅（若 $A, B$ 都可逆） |

### 特殊矩阵

| 矩阵类型 | 定义 | 关键性质 |
|------|------|------|
| **单位矩阵** $I$ | 对角全 1，其余 0 | $AI = IA = A$ |
| **对角矩阵** $\Lambda$ | 非对角元均为 0 | $\Lambda^n$ = 每个对角元的 $n$ 次幂 |
| **对称矩阵** | $A^T = A$ | 不同特征值的特征向量正交 |
| **反对称矩阵** | $A^T = -A$ | 对角元必为 0 |
| **正交矩阵** | $A^T A = I$ | 列向量组是标准正交向量组 |

### 逆矩阵

若 $AB = BA = I$，称 $B$ 为 $A$ 的**逆矩阵**，记 $B = A^{-1}$。

**可逆的充要条件** 🔥：
- $|A| \neq 0$
- $A$ 满秩 —— $\operatorname{rank}(A) = n$
- $A$ 的行（列）向量线性无关
- $AX = 0$ 只有零解
- $A$ 与单位矩阵等价

**求逆矩阵方法** 🔥 🔥：

**伴随矩阵法**（公式）：$A^{-1} = \frac{1}{|A|} \operatorname{adj}(A)$，其中 $\operatorname{adj}(A) = (A_{ji})_{n \times n}$（代数余子式转置排列）。

**初等变换法**（首选）：$[A \mid I] \xrightarrow{\text{初等行变换}} [I \mid A^{-1}]$

> 实际考试中优先选初等变换法——行列式和伴随矩阵元素的计算易错且繁杂。

### 矩阵的秩

**秩** $\operatorname{rank}(A)$ = 矩阵中非零子式的最高阶数 = 行向量组的秩 = 列向量组的秩。

**求秩方法** 🔥：对矩阵做初等行变换化为**行阶梯形**，非零行的行数即为秩。

**秩的性质**（高频）：

- $\operatorname{rank}(A) = \operatorname{rank}(A^T)$
- $\operatorname{rank}(AB) \leq \min(\operatorname{rank}(A), \operatorname{rank}(B))$
- 若 $P, Q$ 可逆，则 $\operatorname{rank}(PAQ) = \operatorname{rank}(A)$
- $\operatorname{rank}(A+B) \leq \operatorname{rank}(A) + \operatorname{rank}(B)$
- 若 $A_{m \times n}$，则 $\operatorname{rank}(A) \leq \min(m, n)$
- **满秩方阵** $\iff$ $|A| \neq 0 \iff A$ 可逆

### 初等变换与初等矩阵

**三种初等行（列）变换** 🔥：
1. 互换两行
2. 某行乘以非零常数 $k$
3. 第 $j$ 行的 $k$ 倍加到第 $i$ 行

每种初等变换对应一个**初等矩阵**（对单位矩阵执行同样变换所得的矩阵）。

> 初等变换**不改变矩阵的秩**，且可以用来求逆矩阵、化矩阵为行最简形。

### 矩阵方程

矩阵方程的求解 🔥：

- $AX = B \Rightarrow X = A^{-1}B$（左乘逆）
- $XA = B \Rightarrow X = BA^{-1}$（右乘逆——注意顺序不能搞反）
- $AXB = C \Rightarrow X = A^{-1} C B^{-1}$

## 向量组与线性方程组

### 向量组的线性相关性

对 $n$ 维向量组 $\alpha_1, \alpha_2, \dots, \alpha_m$，若存在**不全为零**的系数 $k_1, k_2, \dots, k_m$ 使得：

$$
k_1 \alpha_1 + k_2 \alpha_2 + \cdots + k_m \alpha_m = 0
$$

则称该向量组**线性相关**；否则称**线性无关**（即仅当所有系数全为零，上式才成立）。

**线性相关性的判断方法** 🔥：

| 方法 | 判定 |
|------|------|
| 向量个数 $>$ 维数 | **必相关**（最快速的判定） |
| 包含零向量 | **必相关** |
| 两个向量 | 看是否成比例（成比例即相关） |
| 构造矩阵 $A$ 列向量为 $\alpha_i$ | $\operatorname{rank}(A) < 向量个数 \iff $相关；$\operatorname{rank}(A) = 向量个数 \iff $无关 |

**线性相关性的意义**：

- 向量组线性相关 $\iff$ 至少有一个向量可由其余向量线性表出
- 向量组线性无关 $\iff$ 任何一个向量都不可由其余向量线性表出

### 向量组的秩

**极大线性无关组** 🔥：从向量组中挑出的包含向量数最多且仍线性无关的子集。

**向量组的秩** $= 极大线性无关组中的向量个数 = 以这些向量为行（列）构成的矩阵的秩。

> 秩的实质是向量组中「真正独立的方向个数」。

**具体求法**：将向量按列排成矩阵 → 初等行变换化为行阶梯形 → 非零列对应的原向量即为一个极大线性无关组，非零列数 = 秩。

### 线性方程组的解的结构

**齐次线性方程组** $A_{m \times n} \, X = 0$ 🔥：

$$
\text{解的情况：}
\begin{cases}
\operatorname{rank}(A) = n \Rightarrow \text{只有零解} \\
\operatorname{rank}(A) < n \Rightarrow \text{无穷多解（有非零解）}
\end{cases}
$$

**非齐次线性方程组** $A X = b$（$b \neq 0$） 🔥 🔥：

$$
\text{解的情况：}
\begin{cases}
\operatorname{rank}(A) \neq \operatorname{rank}(A \mid b) \Rightarrow \textbf{无解} \\[4pt]
\operatorname{rank}(A) = \operatorname{rank}(A \mid b) = n \Rightarrow \textbf{唯一解} \\[4pt]
\operatorname{rank}(A) = \operatorname{rank}(A \mid b) < n \Rightarrow \textbf{无穷多解}
\end{cases}
$$

> 核心判断口诀：**看秩——无解为秩不等；唯一解为秩相等且等于未知数个数；无穷解为秩相等但小于未知数个数**。

### 基础解系

对 $A X = 0$，若 $\operatorname{rank}(A) = r < n$，则解空间的维数 $= n - r$（未知数个数 $-$ 系数矩阵秩）。

**基础解系** 🔥 就是解空间的任意一组完备基：

- 由 $n - r$ 个线性无关的解向量构成
- 方程组的每一个解都是基础解系的线性组合

**求基础解系步骤** 🔥：
1. 将 $A$ 经过初等行变换化为行最简形
2. 确定自由未知量（共 $n-r$ 个）
3. 分别令每个自由未知量取 $1$，其余自由未知量取 $0$，回代求得基础解系的各个向量

**非齐次方程组通解结构** 🔥：
> **非齐次通解 = 对应齐次方程的通解 + 一个非齐次特解**

**求非齐次特解步骤**：
1. 将增广矩阵 $(A \mid b)$ 化为行最简形
2. 令所有自由未知量取 $0$，回代解出主变量的值——即得一个特解

> 🔥 考研线代解答题大概率最后一步就是写出「通解 = 齐次基础解系的线性组合 + 非齐次特解」。格式规范分一定拿到！

## 特征值与特征向量

### 特征值与特征向量的定义与计算

若 $A$ 为 $n$ 阶方阵，存在数 $\lambda$ 和非零列向量 $x$ 使得：

$$
A x = \lambda x
$$

则 $\lambda$ 为 $A$ 的**特征值**，$x$ 为对应的**特征向量**。

**特征值的求法** 🔥 🔥：

由 $A x = \lambda x \Rightarrow (A - \lambda I) x = 0$，因为 $x \neq 0$，系数矩阵必奇异：

$$
|A - \lambda I| = 0
$$

解这个**特征方程**得到所有特征值 $\lambda_1, \lambda_2, \dots, \lambda_n$。再将每个 $\lambda_i$ 代入 $(A - \lambda_i I)X = 0$，求出其非零解即为该特征值的全体特征向量。

**特征值的性质**（必背）：
- $\sum \lambda_i = \operatorname{tr}(A)$（所有特征值的和 = 矩阵的迹）
- $\prod \lambda_i = |A|$（所有特征值的积 = 行列式值）
- $|A| = 0 \iff A$ 有零特征值（$A$ 奇异）
- 若 $A$ 可逆，$A^{-1}$ 的特征值 = $1 / \lambda_i$（特征向量不变）
- $A^k$ 的特征值 = $\lambda_i^k$（特征向量不变）

### 相似矩阵

若存在可逆矩阵 $P$ 使 $B = P^{-1} A P$，称 $A$ 与 $B$ **相似** 🔥。

**相似矩阵的性质**：
- 特征值完全相同（相似不变量）
- 行列式相同、秩相同、迹相同
- 若 $A \sim \Lambda$（对角阵），则 $\Lambda$ 的对角元即为 $A$ 的所有特征值

**可对角化的条件** 🔥：$n$ 阶方阵 $A$ 可对角化 $\iff$ $A$ 有 $n$ 个线性无关的特征向量。等价的判定条件为：每个 $k$ 重特征值恰有 $k$ 个线性无关的特征向量（即代数重数 = 几何重数）。

> 若 $A$ 有 $n$ 个互不相同的特征值，则 $A$ 必然可对角化——这是充分非必要条件。

对角化步骤：求特征值 → 求每个特征值的特征向量 → $P$ 的列为各特征向量、$\Lambda$ 的对角元为对应特征值。

### 对称矩阵的对角化

**实对称矩阵**（$A^T = A$）具有极强的对角化特性 🔥 🔥：

- 实对称矩阵的各特征值都是实数
- 不同特征值的特征向量彼此**正交**
- 实对称矩阵**必可对角化**——存在正交矩阵 $Q$ 使 $Q^T A Q = \Lambda$

> 正交对角化步骤：求特征值 → 对每个特征值求一组基 — 做 Schmidt 正交化 — 每个正交基的单位化 → 以得到的标准正交特征向量为列构成**正交矩阵 $Q$**。

## 二次型

### 二次型与对称矩阵

含有 $n$ 个变元的二次齐次多项式 $$f(x_1, x_2, \dots, x_n) = \sum_{i=1}^{n} \sum_{j=1}^{n} a_{ij} x_i x_j$$ 称为**二次型**，其中 $a_{ij} = a_{ji}$（对称矩阵 ${A}$ 唯一对应）。

二次型可写成矩阵形式 🔥：

$$
f(X) = X^T A X, \quad \text{其中}\ X = \begin{pmatrix} x_1 \\ x_2 \\ \vdots \\ x_n \end{pmatrix}, \ A^T = A
$$

### 化二次型为标准形

**标准形（只含平方项）**：$f = d_1 y_1^2 + d_2 y_2^2 + \cdots + d_n y_n^2$

**正交变换法** 🔥 🔥：用对称矩阵的正交对角化得到的正交矩阵 $Q$ 做变换 $X = QY$，代入原二次型：

$$
f = X^T A X = Y^T (Q^T A Q) Y = Y^T \Lambda Y = \lambda_1 y_1^2 + \lambda_2 y_2^2 + \cdots + \lambda_n y_n^2
$$

这就是化成了一组**特征值加权**的标准形。

> 🔥 考研线代解答题的核心模板：给二次型 → 写出对称矩阵 $A$ → 求特征向量 Schmidt 正交 → 单位化成正交矩阵 $Q$ → 写出标准形特征值系数组。

**配方法**：逐次提取平方项并消元，与正交变换所得标准形的系数可能不同。

### 正定二次型

二次型 $f(X) = X^T A X$ 若对任意非零向量 $X$ 恒有 $f(X) > 0$，则 $f$ 为**正定二次型**，$A$ 为**正定矩阵** 🔥。

**正定的充要条件**（考研常放在选择题中）：
- 所有特征值 $> 0$
- 各阶顺序主子式全 $> 0$（赛尔维斯特准则）
- $A$ 与 $I$ 合同——即存在可逆矩阵 $C$ 使 $A = C^T C$
- 二次型化标准形的所有平方项系数全为正

> 🔥 判断正定的最快途径：先看各阶主子式主对角元（若主对角元有 $\leq 0$ 则直接判定非正定）。

## 常见题型

- **行列式与矩阵的秩**：该部分为全章的核计基数。每类矩阵先将行列式化为最简而不用过度展开公式。
- **向量组线性相关性的判断**（选择高频）：个数 > 维数 → 立即判相关。看具体向量式子再用内部表出的秩。
- **线性方程组的求解（基础解系）**（大题）：齐次求通解——非齐次增广化行最简——让所有自由元取零回代求特解——写出通解。
- **特征值与特征向量的计算**（大题核心）：解 $|A-\lambda I|=0$ 求各特征值 → $(A-\lambda I)X=0$ 求各线性无关的特征向量。
- **二次型化标准形的正交变换**（高分模板）：写出 $A$ → 计算特征值特征向量 → Schmidt 正交化 → 单位化 → $Q$ 矩阵 → 对角标准形。
