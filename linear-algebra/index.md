# 线性代数（上）— 行列式与矩阵

> **考情提示**：线性代数占数一总分约 22%（≈33 分），通常 2–3 道选择/填空 + 1 道解答题。行列式与矩阵是全部后续章节的工具基础——秩、逆矩阵、初等变换贯穿始终。本章偏重计算，熟练掌握矩阵运算公式即可拿稳基础分。

## 行列式

### $n$ 阶行列式的定义

$n$ 阶行列式是 $n!$ 个项的代数和，每项由不同行不同列的元素乘积配上排列逆序数的正负号组成：

$$
|A| = \det(A) = \sum (-1)^{\tau(p_1 p_2 \cdots p_n)} a_{1p_1} a_{2p_2} \cdots a_{np_n}
$$

> 💀 考研完全不要求用定义硬算 $n$ 阶行列式——所有计算通过性质和初等变换完成。

### 行列式的基本性质（必背）

| 性质 | 说明 |
|------|------|
| 转置不变 | $|A^T| = |A|$ |
| 互换两行（列） | 行列式变号 |
| 两行（列）相等 | 行列式 $= 0$ |
| 某行（列）全零 | 行列式 $= 0$ |
| 某行（列）乘以 $k$ | 行列式变为原来的 $k$ 倍 |
| 某行（列）的 $k$ 倍加到另一行 | 行列式值不变 🔥 |
| 若两行成比例 | 行列式 $= 0$ |
| 上/下三角行列式 | $=$ 主对角线元素乘积 |

### 行列式的计算

**展开定理** 🔥：按第 $i$ 行展开

$$
|A| = \sum_{j=1}^{n} a_{ij} A_{ij}, \quad A_{ij} = (-1)^{i+j} M_{ij}
$$

其中 $A_{ij}$ 为代数余子式，$M_{ij}$ 为余子式。

**初等变换法** 🔥：通过第三种初等行运算（某行的 $k$ 倍加到另一行）将行列式化为**上三角**，值 $=$ 主对角线乘积。

> 考试技巧：首选初等变换化三角行列式——步骤少、计算快、不易出错。只有在变换出现分数极复杂时再用展开法。

**范德蒙德行列式**：

$$
\begin{vmatrix}
1 & 1 & \cdots & 1 \\
x_1 & x_2 & \cdots & x_n \\
x_1^2 & x_2^2 & \cdots & x_n^2 \\
\vdots & \vdots & \ddots & \vdots \\
x_1^{n-1} & x_2^{n-1} & \cdots & x_n^{n-1}
\end{vmatrix}
= \prod_{1 \leq i < j \leq n} (x_j - x_i)
$$

> 只需识别出范德蒙行列式形式后直接背结论代入值——考察频率高但都不过是套公式。

## 矩阵

### 矩阵的基本运算

**加减法**：同型矩阵对应元素相加减。

**数乘**：$kA = (k \cdot a_{ij})$。

**乘法** 🔥：$(AB)_{ij} = \sum_{k=1}^{s} a_{ik} b_{kj}$——$A$ 的列数必须等于 $B$ 的行数，一般 $AB \neq BA$。

**转置**：$(A^T)_{ij} = A_{ji}$。

运算规律（可背）：

| 运算法则 | 是否成立 |
|------|:--:|
| $AB = BA$ | ❌ 一般不成立 |
| $(AB)C = A(BC)$ | ✅ 结合律 |
| $(AB)^T = B^T A^T$ | ✅ 逆转序 |
| $(AB)^{-1} = B^{-1} A^{-1}$ | ✅（若 $A, B$ 都可逆） |

### 特殊矩阵

| 矩阵类型 | 定义 | 关键性质 |
|------|------|------|
| **单位矩阵** $I$ | 对角全 1，其余 0 | $AI = IA = A$ |
| **对角矩阵** $\Lambda$ | 非对角元均为 0 | $\Lambda^n$ = 每个对角元的 $n$ 次幂 |
| **对称矩阵** | $A^T = A$ | 不同特征值的特征向量正交 |
| **反对称矩阵** | $A^T = -A$ | 对角元必为 0 |
| **正交矩阵** | $A^T A = I$ | 列向量组是标准正交向量组 |

### 逆矩阵

若 $AB = BA = I$，称 $B$ 为 $A$ 的**逆矩阵**，记 $B = A^{-1}$。

**可逆的充要条件** 🔥：
- $|A| \neq 0$
- $A$ 满秩 —— $\operatorname{rank}(A) = n$
- $A$ 的行（列）向量线性无关
- $AX = 0$ 只有零解
- $A$ 与单位矩阵等价

**求逆矩阵方法** 🔥 🔥：

**伴随矩阵法**（公式）：$A^{-1} = \frac{1}{|A|} \operatorname{adj}(A)$，其中 $\operatorname{adj}(A) = (A_{ji})_{n \times n}$（代数余子式转置排列）。

**初等变换法**（首选）：$[A \mid I] \xrightarrow{\text{初等行变换}} [I \mid A^{-1}]$

> 实际考试中优先选初等变换法——行列式和伴随矩阵元素的计算易错且繁杂。

### 矩阵的秩

**秩** $\operatorname{rank}(A)$ = 矩阵中非零子式的最高阶数 = 行向量组的秩 = 列向量组的秩。

**求秩方法** 🔥：对矩阵做初等行变换化为**行阶梯形**，非零行的行数即为秩。

**秩的性质**（高频）：

- $\operatorname{rank}(A) = \operatorname{rank}(A^T)$
- $\operatorname{rank}(AB) \leq \min(\operatorname{rank}(A), \operatorname{rank}(B))$
- 若 $P, Q$ 可逆，则 $\operatorname{rank}(PAQ) = \operatorname{rank}(A)$
- $\operatorname{rank}(A+B) \leq \operatorname{rank}(A) + \operatorname{rank}(B)$
- 若 $A_{m \times n}$，则 $\operatorname{rank}(A) \leq \min(m, n)$
- **满秩方阵** $\iff$ $|A| \neq 0 \iff A$ 可逆

### 初等变换与初等矩阵

**三种初等行（列）变换** 🔥：
1. 互换两行
2. 某行乘以非零常数 $k$
3. 第 $j$ 行的 $k$ 倍加到第 $i$ 行

每种初等变换对应一个**初等矩阵**（对单位矩阵执行同样变换所得的矩阵）。

> 初等变换**不改变矩阵的秩**，且可以用来求逆矩阵、化矩阵为行最简形。

### 矩阵方程

矩阵方程的求解 🔥：

- $AX = B \Rightarrow X = A^{-1}B$（左乘逆）
- $XA = B \Rightarrow X = BA^{-1}$（右乘逆——注意顺序不能搞反）
- $AXB = C \Rightarrow X = A^{-1} C B^{-1}$

## 向量组与线性方程组

### 向量组的线性相关性

对 $n$ 维向量组 $\alpha_1, \alpha_2, \dots, \alpha_m$，若存在**不全为零**的系数 $k_1, k_2, \dots, k_m$ 使得：

$$
k_1 \alpha_1 + k_2 \alpha_2 + \cdots + k_m \alpha_m = 0
$$

则称该向量组**线性相关**；否则称**线性无关**（即仅当所有系数全为零，上式才成立）。

**线性相关性的判断方法** 🔥：

| 方法 | 判定 |
|------|------|
| 向量个数 $>$ 维数 | **必相关**（最快速的判定） |
| 包含零向量 | **必相关** |
| 两个向量 | 看是否成比例（成比例即相关） |
| 构造矩阵 $A$ 列向量为 $\alpha_i$ | $\operatorname{rank}(A) < 向量个数 \iff $相关；$\operatorname{rank}(A) = 向量个数 \iff $无关 |

**线性相关性的意义**：

- 向量组线性相关 $\iff$ 至少有一个向量可由其余向量线性表出
- 向量组线性无关 $\iff$ 任何一个向量都不可由其余向量线性表出

### 向量组的秩

**极大线性无关组** 🔥：从向量组中挑出的包含向量数最多且仍线性无关的子集。

**向量组的秩** $= 极大线性无关组中的向量个数 = 以这些向量为行（列）构成的矩阵的秩。

> 秩的实质是向量组中「真正独立的方向个数」。

**具体求法**：将向量按列排成矩阵 → 初等行变换化为行阶梯形 → 非零列对应的原向量即为一个极大线性无关组，非零列数 = 秩。

### 线性方程组的解的结构

**齐次线性方程组** $A_{m \times n} \, X = 0$ 🔥：

$$
\text{解的情况：}
\begin{cases}
\operatorname{rank}(A) = n \Rightarrow \text{只有零解} \\
\operatorname{rank}(A) < n \Rightarrow \text{无穷多解（有非零解）}
\end{cases}
$$

**非齐次线性方程组** $A X = b$（$b \neq 0$） 🔥 🔥：

$$
\text{解的情况：}
\begin{cases}
\operatorname{rank}(A) \neq \operatorname{rank}(A \mid b) \Rightarrow \textbf{无解} \\[4pt]
\operatorname{rank}(A) = \operatorname{rank}(A \mid b) = n \Rightarrow \textbf{唯一解} \\[4pt]
\operatorname{rank}(A) = \operatorname{rank}(A \mid b) < n \Rightarrow \textbf{无穷多解}
\end{cases}
$$

> 核心判断口诀：**看秩——无解为秩不等；唯一解为秩相等且等于未知数个数；无穷解为秩相等但小于未知数个数**。

### 基础解系

对 $A X = 0$，若 $\operatorname{rank}(A) = r < n$，则解空间的维数 $= n - r$（未知数个数 $-$ 系数矩阵秩）。

**基础解系** 🔥 就是解空间的任意一组完备基：

- 由 $n - r$ 个线性无关的解向量构成
- 方程组的每一个解都是基础解系的线性组合

**求基础解系步骤** 🔥：
1. 将 $A$ 经过初等行变换化为行最简形
2. 确定自由未知量（共 $n-r$ 个）
3. 分别令每个自由未知量取 $1$，其余自由未知量取 $0$，回代求得基础解系的各个向量

**非齐次方程组通解结构** 🔥：
> **非齐次通解 = 对应齐次方程的通解 + 一个非齐次特解**

**求非齐次特解步骤**：
1. 将增广矩阵 $(A \mid b)$ 化为行最简形
2. 令所有自由未知量取 $0$，回代解出主变量的值——即得一个特解

> 🔥 考研线代解答题大概率最后一步就是写出「通解 = 齐次基础解系的线性组合 + 非齐次特解」。格式规范分一定拿到！
